A *port* is represented with 16-[[Binary|bit]] **port number**, and they allow different applications to be addressed.

> [!-] [[IPv4|IPv4]] address allow the computer to be communicated with, but *ports* allow specific applications to be communicated with.

> [!conventional-allocations]
> - **Port 80**: [[Hypertext Transfer Protocol|Hypertext Transfer Protocol]]
> - **Port 22**: [[SSH|SSH]]
> - **Port 25**: [[Simple Mail Transfer Protocol|Simple Mail Transfer Protocol]]
> - **Port 53**: [[Domain Name Service|Domain Name Service]]
