A *tristate buffer* allows you to control whether a logic variable is outputted.

| $EN$ | $A$ | $B$  |
| ---- | --- | ---- |
| 0    | X   | Hi-Z |
| 1    | 0   | 0    |
| 1    | 1   | 1    |
Where Hi-Z means high impedance, which means it will not interfere with anything connected to $B$.
