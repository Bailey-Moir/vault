こ refers to something near the speaker, そ refers to something near the listener, and あ refers to something far from both.

| 日本語 | 英語    | Role in Sentence |
| --- | ----- | ---------------- |
| これ  | This  | ╮                |
| それ  | That  | Noun             |
| あれ  | That  | ╯                |
| ここ  | Here  | ╮                |
| そこ  | There | Place            |
| どこ  | Where | ╯                |
| この  | This  | ╮                |
| その  | That  | Qualifier        |
| あの  | That  | ╯                |
