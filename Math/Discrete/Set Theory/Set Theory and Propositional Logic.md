Set theory and propositional logic are closely connected. The following are analogous to one another.

| ST  | PL  |
| --- | --- |
| $\cup$[^1] | $\lor$[^2] |
| $\cap$[^1] | $\land$[^3] |
| $\emptyset$[^4] | $\F$[^5] |
| $U$[^6] | $\T$[^7] |
| $A^c$[^8] | $\neg a$[^9] |

This analogy extends to [[De Morgan's Laws|De Morgan's laws]]:
$$\begin{align*}
	(X \cup Y)^c &= X^c \cap Y^c \\
	(X \cap Y)^c &= X^c \cup Y^c \\
\end{align*}$$
