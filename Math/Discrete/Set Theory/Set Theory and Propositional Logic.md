Set theory and propositional logic are closely connected. The following are analogous to one another.

| ST  | PL  |
| --- | --- |
| $\cup$[^1] | $\lor$^[[[Disjunction]]] |
| $\cap$[^1] | $\land$^[[[Conjunction]]] |
| $\emptyset$^[[[Empty Set]]] | $\F$^[[[Contradictions]]] |
| $U$^[[[Universal Set]]] | $\T$^[[[Tautologies]]] |
| $A^c$^[[[Complement]]] | $\neg a$^[[[Negation]]] |

This analogy extends to [[De Morgan's Laws]]:
$$\begin{align*}
	(X \cup Y)^c &= X^c \cap Y^c \\
	(X \cap Y)^c &= X^c \cup Y^c \\
\end{align*}$$

[^1]: [[Unions and Intersections]]
