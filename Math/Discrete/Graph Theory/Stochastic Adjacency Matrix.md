The only example I've come across is used in [[Page Rank]], where each [[Edges|edge]] on a given [[Vertices|vertex]] has equal [[Probabilities|probability]] of being used. This probability from node $j \to i$ is denoted $M_{ij}$, where
$$M_{ij} = \dfrac{1}{\delta(j)}.$$
Thus, each column sums to $1$, giving a [[Mass Function|probability mass function]].
