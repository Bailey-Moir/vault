If a [[Rings|ring]] $R$ is [[Empty Set|non-empty]], then the *function ring*
$$R^X = \left\{ f \;|\; f : X \to R \right\}$$
with point-wise [[Binary Operations|binary operations]], i.e.
$$\begin{align}
	\left( f+g \right)(x) &= f(x) + g(x) \\
	(f \times g)(x) &= f(x) \times g(x).
\end{align}$$
