A non-zero element $a$ in a [[Groups#^abelian-groups|commutative]] [[Rings|ring]] $R$ is called a *zero-divisor* iff
$$\exists b \in R \setminus \{ 0 \} : ab = 0.$$
> [!proposition] The [[Negation|negation]] of this can be given by $\forall b \in R, \left( ab = 0 \to b = 0 \right)$[^1].

[^1]:  [[Implication from Universal Quantifiers#^contraposing-universal-quantifiers]]
