If $\left| f''(x) \right| \le K$ where $x \in [a,b]$, then
- $\left|E_T\right| \le \dfrac{K(b-a)^3}{12n^2}$^[[[Trapezium Rule]]]
- $\left|E_M\right| \le \dfrac{K(b-a)^3}{24n^2}$ ^[[[Midpoint Rule]]]
Note possible magnitude of $E_M$ is half that of $E_T$.

or if $\left| f^{(4)}(x) \right| \le K$:
- $\left| E_S \right| \le \dfrac{K(b-a)^5}{180n^4}$^[[[Simpson's Rule]]]
