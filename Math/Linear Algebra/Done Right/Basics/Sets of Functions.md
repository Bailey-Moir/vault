If $S$ is a set, then $\mathbb{F}^S$ denotes the set of functions from $S$ to $\mathbb{F}$; the set of functions with a [[MOC Functions|domain]] of $S$ and [[MOC Functions|range]] of $\mathbb{F}$.
$$ \forall \text{sets } S, \left( f \in \mathbb{F}^S \iff f : S \to \mathbb{F}\ \right) $$
> [!note]
> If $S \ne \varnothing$, then $\mathbb{F}^S$ is a [[Vector Spaces|vector space]] over $\mathbb{F}$.
