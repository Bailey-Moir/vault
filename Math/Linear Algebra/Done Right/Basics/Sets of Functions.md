If $S$ is a set, then $\F^S$ denotes the set of functions from $S$ to $\F$; the set of functions with a [[Functions|domain]] of $S$ and [[Math/Algebra/Functions/Functions|range]] of $\F$.
$$ \forall \text{sets } S, \left( f \in \F^S \iff f : S \to \F\ \right) $$
> [!note]
> If $S \ne \varnothing$, then $\F^S$ is a [[Vector Spaces|vector space]] over $\F$.
