A *Bernoulli random variable* is a [[Random Variables|discrete random variable]] with two values: $1$ (success) and $0$ (failure). If $X \sim\text{Bernoulli}(p)$,
- $P(X=1) = p$ then $P(X=0) = 1-p$[^1].
- $\mu_{X} = p$[^2].
